# 0. Update apt

```sh
sudo apt update
sudo apt upgrade
```

# 1. Install zsh.

```sh
sudo apt install zsh -y
```

# 2. Install git

```sh
sudo apt install git -y

# Then, Create Documents/github directory and clone this repository.
```

# 3. Setup

```sh
cd Documents/github/dotfiles_ubuntu

zsh deploy-dotfiles_ubuntu

zsh install.zsh

zsh setup.zsh
```
